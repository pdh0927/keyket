# Keyket
> 여행에 새로운 가치를 더해줄 여행 버킷리스트

✋ 매번 똑같은 패턴의 지겨운 여행은 그만

👨‍👩‍👧‍👦 가족, 친구, 애인과 공유하여 사용하는 버킷리스트

🎢 지역, 테마별로 특색 있는 버킷리스트 항목 추천까지
